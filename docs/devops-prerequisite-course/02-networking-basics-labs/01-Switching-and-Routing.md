1. Which of the following is the first requirement to assign an IP address to a system?
```
The host system must have a physical or virtual network interface. By running ip link command, we can see the interface.
There are multiple interfaces, such as eth, veth, and lo.
```
2. On a Linux-based system, which of the following commands can be used to display the kernel ip routing table?
```
From the `route` command, we can see the route table of the host system.
```

It is a **router**, which helps us to connect two networks together
